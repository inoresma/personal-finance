# Templates de Email para EmailJS

Debes crear estos templates en tu cuenta de EmailJS:
https://dashboard.emailjs.com/admin/templates

## Instrucciones

1. Ve a EmailJS Dashboard
2. Haz clic en "Email Templates" en el menú lateral
3. Haz clic en "Create New Template"
4. Copia el contenido HTML de cada archivo .html a tu template
5. Configura el nombre del template según se indica (Template ID)

## Templates a crear:

### 1. debt_reminder
- **Template ID:** `debt_reminder`
- **Archivo:** `debt_reminder.html`
- **Uso:** Recordatorios de deudas próximas a vencer

### 2. budget_alert
- **Template ID:** `budget_alert`
- **Archivo:** `budget_alert.html`
- **Uso:** Alertas de presupuesto bajo o excedido

### 3. monthly_summary
- **Template ID:** `monthly_summary`
- **Archivo:** `monthly_summary.html`
- **Uso:** Resumen mensual de finanzas

## Variables disponibles

Cada template usa variables con la sintaxis `{{variable_name}}`.
Las variables están documentadas en cada archivo HTML.
